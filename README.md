# Todo List Web App
A clean and responsive To-Do List website built with HTML, CSS and JavaScript.

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Dark mode
- Search tasks
- Sort tasks
- Category filters
- Priority levels
- Due dates
- Progress tracking
- Local storage persistence

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

css/
js/
index.html

## Future Improvements

- Drag and drop tasks
- Toast notifications
- Export/Import tasks
- Keyboard shortcuts
- Calender View
